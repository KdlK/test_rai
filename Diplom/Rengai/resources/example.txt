База знаний Rengai - Примеры использования

=====================================
Основные команды Renga
=====================================

1. Открытие Renga:
   - Запустите Renga через ярлык или файл .renga
   - При первом запуске создаётся новый проект

2. Создание стен:
   - Перейдите на вкладку "Стены"
   - Выберите инструмент "Стена"
   - Укажите начальную и конечную точку
   - Задайте высоту и толщину

3. Создание перекрытий:
   - Вкладка "Перекрытия"
   - Инструмент "Перекрытие"
   - Обведите контур

4. Создание колонн:
   - Вкладка "Колонны"
   - Разместите колонну по одной или массивом

=====================================
Работа с уровнями
=====================================

- Создание уровня: меню "Уровни" -> "Создать уровень"
- Удаление уровня: правый клик -> "Удалить"
- Переименование: двойной клик по имени

=====================================
Измерения
=====================================

- Объём стены: выбрать стену -> "Свойства" -> "Объём"
- Площадь перекрытия: выбрать перекрытие -> "Свойства" -> "Площадь"
- Высота этажа: свойства уровня

=====================================
Сохранение проекта
=====================================

- Сохранить: Ctrl+S
- Сохранить как: Ctrl+Shift+S
- Экспорт в DWG: Файл -> Экспорт
