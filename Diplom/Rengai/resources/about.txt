# Renga - информация о программе

Renga — это российская система архитектурно-строительного проектирования (BIM).

## Основные возможности

- Создание 3D моделей зданий
- Работа с архитектурными элементами: стены, перекрытия, колонны, окна, двери
- Экспорт в IFC формат
- Создание чертежей и спецификаций

## Категории элементов

- Стены (Walls) — вертикальные ограждающие конструкции
- Перекрытия (Floors) — горизонтальные конструкции между этажами
- Колонны (Columns) — вертикальные несущие элементы
- Окна и двери

## Как получить объём

Для подсчёта объёма элементов в Renga используется API автоматизации.
