## Файлы проекта

### Rengai/
- `main.py` - Основное приложение (PyQt6) - полностью переписан 20.04.2026
- `config.py` - Конфигурация + чтение ресурсов + read_all_resources()
- `llm_client.py` - Клиент для Ollama
- `context.py` - Управление историей чата
- `skills/calculate_volume.py` - Скилл расчёта объёма с detect_category_from_text()
- `resources/` - База знаний (about.txt, PDF файлы)

### renga_api/
- `core/app.py` - Подключение к Renga через COM
- `core/typelib.py` - Пути к TLB файлам
- `skills/model.py` - Работа с моделью
